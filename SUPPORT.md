# Support

GitHub Monitor is maintained as a small open source utility.

Use GitHub issues for reproducible bugs, compatibility problems, and focused feature requests. Please include your Node.js version, operating system, browser, and whether you are using `GITHUB_TOKEN`, `GH_TOKEN`, or GitHub CLI fallback auth.

Out of scope:

- support for hosted multi-user deployments;
- debugging private repositories without a minimal reproduction;
- handling or storing GitHub credentials;
- guaranteed response times.
